Liveability Index

Data from the Economist Intelligence Unit

Overall rating, from 100 (ideal) to 1(intolerable)

More information

This tool has beeen designed to allow the easy exploration of data; viewing each city either by their ranking, their geographical location or the individual components that make up the cities score.

Hover over the points on the map or the bar chart to activate the radial area plot. Then hover over the radial area plot to view what each point represents.

The main purpose of this design has been to make it easy and fun to explore this dataset, so bearing this in mind have fun with the data!
